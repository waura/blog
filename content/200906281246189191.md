Git Extensionsから[SSH](http://d.hatena.ne.jp/keyword/SSH)経由でgitを使うために[WinSCP](http://d.hatena.ne.jp/keyword/WinSCP)についてくるPuttyGen等で作成する。
Git Extensionsで使用する[ssh](http://d.hatena.ne.jp/keyword/ssh)のppkキーはssh1のものでないとエラーがでる。
 Urlはこんな感じで指定する。
```
[ssh](http://d.hatena.ne.jp/keyword/ssh)://@:/var/repos/hogehoge.git
```
